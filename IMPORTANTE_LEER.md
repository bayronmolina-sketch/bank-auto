Este es un programa automatizado para descargar cartolas de la empresa Nutrisco.

Pasos para ejecutar:
1- Descarga prp-py
2- abre el archivo con visual studio code
3- Crea las carpetas: CARTOLAS BANCARIAS\BCI (el programa creara solo las carpetas de las empresas).
4- abre cmd
5- ejecuta cd Desktop
6- ejecuta cd prp-py
7- ejecuta py bci-cartolas.py

**Recomendado: desactivar la opcion de abrir automaticamente adobe si se descarga pdfs**
En caso de algun error pruebe ejecutando el archivo pruebas.py con los pasos de arriba.
